# 给集成团队的说明

Atlas 被设计为跨多个机器人项目的通用传感器基础设施。

一旦完成集成，Atlas 可作为以下系统的确定性同步骨干：

*   感知传感器栈
*   导航系统
*   机器人研究平台
*   生产型机器人集群

建议评估 Atlas 的团队从阅读 **《Atlas 评估套件快速入门指南》** 开始，随后参考 **硬件集成包** 进行机械结构和电气性能验证。

对于面向生产计划的项目，团队还应审阅 **OEM 集成包** 以及未来发布的相关 **认证与生产就绪文档**。
